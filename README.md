# React Blog Website

This is a modern blog website built using React and Vite. It includes multiple pages like Home, Blog, About, and Contact with a clean UI design.

## 🚀 Features
- Responsive design
- Multiple pages (Home, Blog, About, Contact)
- Reusable components
- Fast performance with Vite

## 🛠️ Technologies Used
- React
- Vite
- CSS

## 📦 Installation

Clone the repository:

git clone https://github.com/abdullah-az-jobayerr/react-blog.git

Go to the project directory:

cd react-blog

Install dependencies:

npm install

Run the development server:

npm run dev
